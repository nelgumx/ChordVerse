# Simple Chords — Full Chord Diagram Edition

This version includes an editor, automatic chord-line detection, and hover/tap guitar diagrams.

## Chord diagram coverage

The diagram engine supports all 12 chromatic roots, written with sharps or flats:

- C, C# / Db
- D, D# / Eb
- E
- F, F# / Gb
- G, G# / Ab
- A, A# / Bb
- B

It also generates diagrams for common chord families:

- Major and minor
- 5/power chords
- 6 and minor 6
- 7, major 7, and minor 7
- 9 and add9
- sus2 and sus4
- diminished and augmented
- Slash chords display the diagram for the main chord

## Run the website

Open `index.html` in your browser. Hover over a chord with a mouse, focus it with the keyboard, or tap it on a phone.
